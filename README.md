## ✨ **Why DevQuest?**
- 🗺️ **Interactive Roadmaps** — Visual paths for **Frontend**, **Backend**, **DevOps**, **AI/ML**, & more.  
- 🎯 **Daily Quests** — Personalized challenges, streaks, XP, & epic achievements.  
- 📚 **Curated Resources** — Hand-picked tutorials, community tips, & smart search.  
- 👥 **Community** — Study groups, leaderboards, and collab learning.  
- 📊 **Progress Dashboard** — Analytics to track your mastery.

---

## 🛠️ **Tech Stack**
**Frontend:** Next.js 14 + React + TypeScript  
**UI:** Tailwind CSS + shadcn/ui  
**Auth & DB:** Supabase (PostgreSQL)  
**Deploy:** Vercel 🚀
